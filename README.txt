When getting started with the MIL Electrical Team please go through the following steps...

1) Go through the EE Git Setup.docx file and make sure that you're github environment is setup properly
   on your PC.

2) Go through the Training_Altium documents to learn how to design PCBs.
   You can find the link to the repo here: https://github.com/uf-mil-electrical/Training_Altium
   

3) Go through the Training_Firmware documents to learn how to design firmware at MIL.
   You can find the link to the repo here: https://github.com/uf-mil-electrical/Training_Firmware

NOTE: Steps 2 and 3 can be done interchangeably depending on what your task is.
